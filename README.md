# 🌿 Prakriti Herbs HRMS Portal

A complete enterprise HRMS system for managing employees, attendance, payroll, and operations.

---

## 🚀 Features

### ✅ Attendance System
- GPS आधारित attendance
- Face Scan attendance (photo proof)
- Fingerprint support
- Office geofence restriction
- Kiosk mode

### 📊 Dashboard
- Real-time staff status
- Present / Late / Absent tracking
- Monthly insights

### 👨‍💼 Employee Management
- Add / edit / delete employees
- Role-based system
- Shift & timing management

### 📅 Leave Management
- Apply / approve leaves
- Paid & casual leave tracking

### 💰 Payroll
- Automatic salary calculation
- Late penalty system
- Monthly reports

### 📁 Document Verification
- Upload & manage documents
- Verification status tracking

### 📢 Notice Board
- Send notices to staff
- Inbox / Sent system

### 📍 Office Location
- GPS geofence setup
- Branch management

### 🗑️ Trash System
- Soft delete
- Auto delete after 30 days

---

## 🔐 Authentication

- Login: ID + Password
- OTP: Only for Forgot Password
- JWT based security

---

## ⚙️ Tech Stack

**Frontend:**
- React + Vite
- Tailwind CSS

**Backend:**
- Node.js + Express

**Database:**
- PostgreSQL
